## File organization helper

If files become messy, double-click:

```text
ORGANIZE_FILES.bat
```

This creates a clean research folder layout under `projects/` and moves loose user files from the root folder into:

```text
projects/_inbox/YYYY-MM-DD/
```

It does not delete files.
It does not move important environment folders such as `tools/`, `docs/`, `projects/`, `vscode/`, or `winpython/`.

Recommended beginner rule:

```text
If you do not know where to put a file, put it in the folder and run ORGANIZE_FILES.bat.
```
