# 初心者向け AI カタログの使い方

## 基本

Python や .venv が分からなくても大丈夫です。
基本的には `.bat` ファイルをダブルクリックして使います。

## YOLO の使い方

1. `YOLO_INSTALL.bat` をダブルクリックします。
2. インストールが終わるまで待ちます。
3. `YOLO_RUN.bat` をダブルクリックします。
4. `projects/yolo-sample/yolo_result.jpg` が作成されれば成功です。

## VS Code の注意

VS Code の右上の再生ボタンは、違う Python を使ってしまうことがあります。
初心者は、まず `RUN_*.bat` またはルートの `*_RUN.bat` を使ってください。

## エラーが出たとき

黒い画面を閉じずに、表示されたエラーを担当者に見せてください。
