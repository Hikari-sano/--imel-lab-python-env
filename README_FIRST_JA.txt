Mimel Lab Python / AI 環境の使い方

最初にやること:

1. Start.bat をダブルクリックします。
   Python と VS Code が自動で準備されます。

2. AI を使いたい場合は Install-AI.bat をダブルクリックします。
   初心者は、まず YOLO_INSTALL.bat を使うのがおすすめです。

3. YOLO を試す場合:
   (1) YOLO_INSTALL.bat をダブルクリック
   (2) 終わったら YOLO_RUN.bat をダブルクリック

注意:
- VS Code右上の再生ボタンを押すと、違うPythonで実行されることがあります。
- 初心者は、各プロジェクト内の RUN_*.bat またはルートの *_RUN.bat を使ってください。
- AI系の初回インストールは時間がかかります。
- エラーが出たら、黒い画面を閉じずに担当者へ見せてください。
